SEO och nyckelord

From SEO perspective, using relevant keywords in the page title, heading, meta description and content is very important as Google will index your page by these keywords.

What is the keywords strategy? First, we can check Google Trends about the hottest topic. Second, we can make a keyword list and check in Keywords Planner or other tools about their competence and monthly search volumn. Third, sort by search volume(from high to low) and competition(from low to high). Those keywords with high search volume and relevant lower competence are the chance that you may get high rank at Google

Responsive Webbdesign

Responsive design means that the page layout can be adjusted smoothly according to different viewport so that visitors can have best user experience no matter if they use pc,tablet or mobile phone to browse your webiste.

Several methods that people can use to make the page responsive: 1) Setting responsive meta tag; 2) Use grid-view; 3)Use Media Query

Brypunkter means that you should indiciate different style for different screen size, it should be like this: @media all and (max-width: 800px) { .navigation-container {}}

Network

How network works? When we key in a URL in the browser, our ISP will contact DNS first,then DNS will give back an IP address of this webiste, then we use IP address to contact the server of the website, then they will return back the relevant page content.

Bootstrap

Bootstrap is a CSS library which provides stylesheet that can make your page responsive.

There are two ways that people can use bootstrap: 1)download the relevant bootstrap library under relevant directory and make a link at the head element. 2)link the bootstrap url at the head element directly. The difference between them is, for the first method, you can use bootsrap sytle offline while for the second method you can only use these styles when you are online.
